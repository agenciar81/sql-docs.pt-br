# <a name="getinstancename-method-sqlserverdatasource"></a>Método getInstanceName (SQLServerDataSource)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Retorna o nome da instância do [!INCLUDE[ssNoVersion](../../../includes/ssnoversion-md.md)].  
  
## <a name="syntax"></a>Sintaxe  
  
```  
  
public java.lang.String getInstanceName()  
```  
  
## <a name="return-value"></a>Valor retornado  
 Uma **String** que contém o nome da instância ou o valor nulo no caso de nenhum valor ter sido definido.
