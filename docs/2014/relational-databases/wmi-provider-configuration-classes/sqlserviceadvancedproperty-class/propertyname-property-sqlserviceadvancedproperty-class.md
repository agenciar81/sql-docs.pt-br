# <a name="propertyname-property-sqlserviceadvancedproperty-class"></a>Propriedade PropertyName (classe SqlServiceAdvancedProperty)
  Obtém ou define o nome de uma propriedade avançada.  
  
## <a name="syntax"></a>Sintaxe  
  
```  
  
object  
.PropertyName [= value]  
```  
  
## <a name="parts"></a>Partes  
 *objeto*  
 Um objeto da [classe SqlServiceAdvancedProperty](sqlserviceadvancedproperty-class.md) que representa uma propriedade avançada.  
  
## <a name="property-valuereturn-value"></a>Valor da propriedade/Valor do retorno  
 Um valor da cadeia de caracteres que especifica o nome da propriedade avançada.
