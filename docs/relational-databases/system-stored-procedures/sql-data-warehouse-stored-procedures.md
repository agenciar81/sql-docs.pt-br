##  <a name="AggregateFunctions"></a>[sp_datatype_info_90 &#40;SQL data warehouse&#41; ](../../relational-databases/system-stored-procedures/sp-datatype-info-90-sql-data-warehouse.md)

 [sp_pdw_database_encryption_regenerate_system_keys &#40;SQL data warehouse&#41;](../../relational-databases/system-stored-procedures/sp-pdw-database-encryption-regenerate-system-keys-sql-data-warehouse.md)

> [!NOTE]  
>  Alguns procedimentos armazenados adicionais do sistema são usados somente dentro de uma instância do [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] ou por meio de APIs de cliente e não se destinam ao uso geral do cliente. Esses procedimentos estão listados em [procedimentos armazenados do sistema (Transact-SQL)](https://msdn.microsoft.com/library/ms187961.aspx). Esses procedimentos estão sujeitos a alterações e a compatibilidade não é garantida. Todos os procedimentos da lista não estão disponíveis no [!INCLUDE[ssSDW](../../includes/sssdw-md.md)].
