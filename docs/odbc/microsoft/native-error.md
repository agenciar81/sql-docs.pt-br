# <a name="native-error"></a>Erro nativo
Códigos de erro nativos positivos são gerados pelos drivers do banco de dados da área de trabalho. Códigos de erro negativos são gerados pelo Microsoft Jet. Para obter mais informações, consulte o *Guia do programador do Microsoft Jet mecanismo de banco de dados*.
