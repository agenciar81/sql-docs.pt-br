# <a name="odbc-errors"></a>Erros de ODBC
Quando ocorre um erro, os drivers de banco de dados do Microsoft ODBC Desktop retornam o número do erro nativo, o SQLSTATE (um código de erro ODBC) e uma mensagem de erro.  
  
 Esta seção contém os seguintes tópicos:  
  
-   [Erro nativo](../../odbc/microsoft/native-error.md)  
  
-   [SQLSTATE](../../odbc/microsoft/sqlstate.md)  
  
-   [Mensagens de erro](../../odbc/microsoft/odbc-jet-error-messages.md)  
  
-   [Erros aritméticos](../../odbc/microsoft/arithmetic-errors.md)
