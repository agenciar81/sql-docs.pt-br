# <a name="sqlfetch-visual-foxpro-odbc-driver"></a>SQLFetch (Driver ODBC do Visual FoxPro)
> [!NOTE]  
>  Este tópico contém informações específicas do driver ODBC do Visual FoxPro. Para obter informações gerais sobre essa função, consulte o tópico apropriado em [referência da API do ODBC](../../odbc/reference/syntax/odbc-api-reference.md).  
  
 Suporte: completo  
  
 Conformidade da API ODBC: nível de núcleo  
  
 Recupera uma linha de um conjunto de resultados para os locais especificados pelas chamadas anteriores para [SQLBindCol](../../odbc/microsoft/sqlbindcol-visual-foxpro-odbc-driver.md). Prepara o driver para uma chamada para [SQLGetData](../../odbc/microsoft/sqlgetdata-visual-foxpro-odbc-driver.md) para as colunas desassociadas.  
  
 Para obter mais informações, consulte [SQLFetch](../../odbc/reference/syntax/sqlfetch-function.md) na *referência do programador de ODBC*.
